# Sales Forecasting and Business Analytics using Python

## Project Overview

This project analyzes retail store sales data and predicts future sales using Machine Learning. The dataset contains 9,800 customer orders from a superstore. The project includes data analysis, visualization, and sales forecasting using Linear Regression.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn
* Jupyter Notebook

## Dataset Information

* Total Orders: 9,800
* Total Revenue: $2,261,536.78
* Average Order Value: $230.77
* Highest Single Sale: $22,638.48

## Key Insights

### Category Analysis

* Technology generated the highest revenue.
* Office Supplies had the highest number of orders.

### Region Analysis

* West region generated the highest revenue.
* South region generated the lowest revenue.

### Product Analysis

* Canon imageCLASS 2200 Advanced Copier was the highest revenue-generating product.

### Sales Forecasting

A Linear Regression model was trained on monthly sales data to forecast future sales. The model predicts a gradual increase in sales over the next few months.
